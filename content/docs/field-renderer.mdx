## Usage

```tsx
import { FieldRenderer } from "@/components/ui/field-renderer";
```

```tsx
<FormField
  control={form.control}
  name="username"
  render={({ field }) => (
    <FieldRenderer label="Username">
      <Input placeholder="username" {...field} />
    </FieldRenderer>
  )}
/>
```

## Accepted Props

| Prop          | Type              | Description                                      |
| ------------- | ----------------- | ------------------------------------------------ |
| `label`       | `React.ReactNode` | The label displayed above the input field.       |
| `description` | `React.ReactNode` | Optional description text below the input field. |
| `children`    | `React.ReactNode` | The input element or any additional children.    |

## Dependencies

This component depends on the `Form` component from shadcn/ui. Ensure you have it installed and set up correctly.

[View Form Component Documentation](https://ui.shadcn.com/docs/components/form)

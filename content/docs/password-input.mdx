## Installation

Add this in your global css file.

```css
/* hide default eye icon on password field  */
input[type="password"]::-ms-reveal,
input[type="password"]::-ms-clear,
input[type="password"]::-webkit-contacts-auto-fill-button,
input[type="password"]::-webkit-credentials-auto-fill-button {
  display: none;
}
```

## Usage

```tsx
import { PasswordInput } from "@/components/ui/password-input";
```

```tsx
<FormField
  control={form.control}
  name="password"
  render={({ field }) => (
    <FieldRenderer label="Password">
      <PasswordInput placeholder="password" {...field} />
    </FieldRenderer>
  )}
/>
```

## Dependencies

This component depends on the `Form` component from shadcn/ui. Ensure you have it installed and set up correctly.

[View Form Component Documentation](https://ui.shadcn.com/docs/components/form)

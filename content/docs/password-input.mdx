## How To Use

Using this component is as simple as working with any standard input field in a form. After installation, simply import the component and place it within your form structure.

<Callout title="Usage Restriction">
  This component is designed to work exclusively within the Form component
  provided by shadcn/ui. Make sure you wrap it inside a valid Form context to
  ensure proper functionality.
</Callout>
<Steps>
<Step>
### Copy and paste this css code in your `global.css`file.

```css
/* hide default eye icon on password field  */
input[type='password']::-ms-reveal,
input[type='password']::-ms-clear,
input[type='password']::-webkit-contacts-auto-fill-button,
input[type='password']::-webkit-credentials-auto-fill-button {
  display: none;
}
```

</Step>
<Step>
### Run this command in your terminal to add this component in your project.

<PackageInstall name={'password-input'} />
</Step>
<Step>
### Import this component where you want to use it.

```tsx
import { PasswordInput } from '@/components/ui/password-input';
```

</Step>
<Step>
### Add this component inside `FormField` in the place of `Input` component.

```tsx
<FormField
  control={form.control}
  name="password"
  render={({ field }) => (
    <FieldRenderer label="Password">
      <PasswordInput placeholder="password" {...field} />
    </FieldRenderer>
  )}
/>
```

</Step>
</Steps>
